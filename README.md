# Modern Calculator

A clean, modern calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clear (C) and delete (←) buttons
- Decimal point support
- Responsive design
- Modern glassmorphism UI

## How to Use

1. Open `index.html` in any web browser
2. Use the calculator as you would a standard calculator
3. Click buttons to perform calculations

## File Structure

- `index.html` - HTML structure
- `style.css` - Styling with modern glassmorphism design
- `script.js` - Calculator logic and functionality

## Design Features

- Glassmorphism effect with blurred background
- Responsive layout for different screen sizes
- Smooth button animations
- Color-coded buttons for different functions
- Clear display with easy-to-read numbers

## Browser Support

Works in all modern browsers that support CSS3 and ES6 JavaScript.

## Customization

You can easily modify the colors in `style.css`:
- Change the gradient in the body background
- Adjust button colors for different themes
- Modify the glassmorphism effect by changing the backdrop-filter and background opacity

## Development

To modify the calculator:
1. Edit `index.html` to change the button layout or structure
2. Update `style.css` for styling changes
3. Modify `script.js` to change the calculator logic

Enjoy your modern calculator!
